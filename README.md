# Truth Table Minimizer
## Quine-McClusky Minimization Implementation

### Compile
```
g++ -std=c++11 -Wall -Wextra -o minimize main.cpp minimizer.cpp minterm.cpp
```

### Dependencies
```
    main.cpp

    minterm.cpp
    minterm.h

    minimizer.cpp
    minimizer.h
```

### Usage
```
./minimize [n-bit string] ... where n is a power of 2
```
